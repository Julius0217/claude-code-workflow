# MCP 서버 설정 예제

Model Context Protocol (MCP) 서버 설정 예제입니다.

## MCP 개요

MCP는 Claude Code가 외부 시스템과 상호작용할 수 있게 해주는 프로토콜입니다.

## 설정 위치

```
~/.claude/settings.json
```

## 예제 목록

1. [database-servers.json](database-servers.json) - 데이터베이스 연동
2. [browser-automation.json](browser-automation.json) - 브라우저 자동화
3. [file-system.json](file-system.json) - 파일 시스템 확장
4. [custom-api.json](custom-api.json) - 커스텀 API 연동

## 전체 설정 예제

[full-config.json](full-config.json)에서 여러 MCP 서버를 조합한 전체 설정을 확인할 수 있습니다.
